# 人物信息
外麻属于一个泛指，可以量产。
但是关键的一些外麻会被单独取名字。
## 性别
女，没有例外。
## 身高
`145~165cm`，没有例外。
## 相关信息
这是[麻](<./麻.md>)在外界的替身。其本身可以自律，但是更多时候[^1]是麻在操控。
[^1]:真的不是是大部分都没有在控制而是在摸鱼吗？
自律的性能相当地好，其意识为群体意识，这是[kps-020 沫芣·流飒](<../编号/kps/kps-020 沫芣·流飒.md>)做的第一个“信息池”。
为什么身高有差异呢？
⏺️录音（来自～[麻](<./麻.md>)与[kps-000 岁依璃伊](<../编号/kps/kps-000 岁依璃伊.md>)的对话）
```
麻：身高这种东西完全没有必要去考虑嘛。
    你看，高一点矮一点又没有什么关系。
    （抬头）
    （伸手比了比）
    （沉默）
依璃伊：是的呀，完全没有必要再长高了呢。
    （微笑）
    （摸摸头）
    这样的麻已经很可爱了耶。
麻：……
```
注意到依璃伊的身高在`163cm`左右徘徊，于是麻制造外麻时稍微造成了一些误差，导致外麻的身高可能会高于依璃伊。
