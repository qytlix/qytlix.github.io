### 启动conda
```sh
conda init zsh
```
### 查看环境
```sh
conda env list
```
### 激活环境
```sh
conda activate <name>
```
### 关闭conda
打开`.zshrc`删掉
```zsh
# >>> conda initialize >>>
# ...
# <<< conda initialize <<<
```
